# msp432-LaunchPad

## 代码库资源介绍

|    .c文件目录    |      功能       |
| :--------------: | :-------------: |
|     delay.c      | 基本的延时功能  |
|      EXIT.c      |    外部中断     |
|      isr.c       |  中断服务函数   |
|      key.c       |  板载按键扫描   |
|      led.c       |     板载led     |
|     MyGPIO.c     |    gpio配置     |
|     myuart.c     |    串口配置     |
|      OLED.c      |  IIC通信的oled  |
|    OLED_SPI.c    |  SPI通信的oled  |
|     servo.c      |    舵机配置     |
| soft_interrupt.c |  软件中断配置   |
|      sys.c       |    时钟配置     |
|     TimerA.c     |   定时器A配置   |
|      VOFA.c      | vofa+上位机功能 |
